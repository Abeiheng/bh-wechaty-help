# bh-wechaty-help

## 项目介绍

北恒微信助手，解放您的双手，多种方案可同时运行

- [x] 好友申请
- [x] 个人回复
- [x] 群聊回复
- [x] 消息转发
- [x] 屏蔽检测
- [x] 群发消息
- [x] 日志列表

此项目基于 **Wechaty** 以及 **Electron** 进行编写：<https://wechaty.gitbook.io/wechaty/v/zh/>
